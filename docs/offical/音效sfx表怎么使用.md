# 音效sfx表怎么使用(sfx)
### 音效分类
1. 卡牌入槽时卡牌音效配置 (sfx_npc_role_dub.json)
2. 仪式结算新出现卡牌音效配置(sfx_settle_card_new.json)
### 卡槽卡牌放入音效配置
需要新增或修改 卡牌放置到仪式(rite)卡槽的音效, 可以在mod的config目录下新建sfx_npc_role_dub.json文件, 按下列基本结构要求填写内容.
基本结构说明
```json
{
	// 卡牌id, 对应备选音效名称, 随机播放
    "2000001": ["sfx01", "sfx02", ...]
}
```
其中 key 为卡牌id, value为一个数组, 对应于sfx目录下的具体音效文件名(不包含后缀), 当卡牌放置到仪式卡槽中时, 会触发该列表中随机一个音效的播放
### 仪式结算新卡牌音效配置
需要新增或修改 卡牌在仪式结算中获得的音效, 可以在mod的config目录下新建 sfx_settle_card_new.json文件, 按下列基本结构要求填写内容.
基本结构说明
```json
{
    // 通用音效（非角色卡默认使用通用音效）
	"0": "settle_card_new_nomal", 
	// 正面音效（角色卡默认使用正面音效）
    "1": "settle_card_new_great", 
    
    // 谗言
    "2000168": "settle_card_new_bad", 
}
```
其中 key 为卡牌id, value 为 sfx目录下 具体音效文件名(不包含后缀), 当卡牌在仪式结算被生成时, 会触发当前音效. 
特别的, 如果对应卡牌不存在指定的音效配置, 则会基于其卡牌类型, 播放默认的 0 号(默认音效) 或 1 号(角色卡牌)音效
