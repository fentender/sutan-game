# 如何修改，新建一个结局（over）
### 结局（over）说明
- 涉及游戏的结局的cg，描述和后日谈
- 激活某个结局，代表当前游戏已结束
### 如何新建一个结局（over）
- 所有结局都配置在over.json表中
- 在over表中新建未被使用的1~999数字来作为结局的编号
### over表的数据结构说明
```json
{
    "1": { //结局的编号，新建未被使用的1~999
        "name": "伴君如伴虎", //结局名称
        "sub_name": "你死于触怒苏丹", //结局名称下的结局的一段简短描述
        "text": "这并不有趣，你被苏丹像小丑一样处死。", //结局的详细描述
        "text_extra": [ //允许设计者，设计不同条件下显示不同额外的结局描述
            {
                "condition": {
                },
                "result_text": ""
            } //需要注意：即使没有额外的结局描述，这段参数也不能删除
        ],
        "open_after_story": 0, //是否开启后日谈，配置为1为开启，配置为0为关闭
        "bg":"over_cg/over_cg_1", //结局的cg资源
        "icon":"over_icon/over_icon_1", //结局弹框的UI资源，建议不要修改
        "title":"over_icon/over_title", //结局弹框的UI资源，建议不要修改
    },
    ,
    "222": {
        "name": "追忆之国",
        "sub_name": "从此再没有苏丹了，只有复辟伟大故国的伟大国王！国王万岁！",
        "success": 2, 
        //额外参数，用于标识结局是失败，小胜还是大胜
        //如果配置为0或者没有这段参数，意味着是失败
        //如果配置为1，意味着是小胜
        //如果配置为2，意味着是大胜
        "text": "你和希尔希纳一起踏上了青金石宫廷。.......",
        "text_extra": [
            {
                "condition": {
                },
                "result_text": ""
            }
        ],
        "open_after_story": 1,
        "bg":"over_cg/over_cg_101",
        "icon":"over_icon/over_icon_1",
        "title":"over_icon/over_title",
        "manual_prompt":true
        //额外参数，用来开关触发结局时是否弹出确认弹框
        //失败结局默认不加这段参数或者配置为false，实际会表现为弹出“是否会回到本回合开始”的确认弹框
        //成功结局都需要配置这段参数并配置为true，用来关闭确认弹框
    },
  }
```
### 后日谈说明
- 当触发的结局开启了后日谈，系统将从所有后日谈配置检索满足条件后日谈播放
- 后日谈配置和角色绑定
### 如何新建一个后日谈
- 在mod提供的地址中找到对应角色的后日谈文件
- 在对应的角色文件内，添加带有唯一编号的后日谈配置
- 文件id和角色并非强绑定，只是策划方便管理。实际上你可以新建一个未被使用的id，将任意多个的角色的后日谈配置到这一个文件中
    - 只要触发条件配置正确即可
![图片](https://docimg5.docs.qq.com/image/AgAABS6GRkfG7E2KDUdJKKrs4ws2AZ7E.png?w=664&h=527)
### 后日谈数据结构说明，以2000019热娜为例
```json
{
    "id": 2000019, //热娜的卡牌编号
    "name": "热娜", //热娜的名称
    "prior": [
    ],
    "extra": [
//---------------------------结局-屠龙-神圣的牺牲------------------------------
        {
            "key": "2000019_extra_1", 
            //后日谈唯一编号，为保证不会和出现重复的，建议用[角色id_extra_数字编号]组合
            "condition":{ 
            //因为后日谈是分角色配置的，所以在条件中需要完整描述对应结局已经触发的条件
            //如果没有配置完整，会导致某个后日谈不触发或者在错误的结局中触发
                //--------造反结局条件---------------------
                "counter.7000550>=":1, //完成神圣的牺牲结局
            //--------个人条件---------------------     
                "have.2000019":1, //热娜存活
            },
            "result_text": "热娜以屠龙为灵感做了一件传世的珠宝，....." //后日谈文本描述
        },
//---------------------------结局贤者之国-----------------------
        {
            "key": "2000019_extra_2",
            "sort" :1, 
            //当前后日谈的显示顺序，数字越大越靠后
            //如果不加这段参数，就默认按照角色id的数字顺序显示
            "pic":"cards/2000019",
            //显示后日谈插图
            //如果不加这段参数默认用当前角色的立绘
            "condition":{
                //--------造反结局条件---------------------
                "counter.7000449<":1, //主角参与果实
                "have.2000910.正当性":1, //革命的计划
                "have.2000312.苏丹":1,  //奈费勒成为了领袖
                "counter.7000446>=":1, //平衡
            //--------个人条件--------------------- 
                "have.2000019":1,
            },
            "result_text": "你的议院通过了一项贵族晋升制度......"
        }, 
    ]
}
```
